# myDevEnv
## Versions
- node v8.9.4
- npm 5.6.0
## Install
```
$ npm i
```

### if you are going to postcss
```
$ npm i
$ npm uninstall gulp-sass gulp-pleeease gulp-csscomb -D
$ npm i gulp-postcss autoprefixer postcss-nested postcss-import postcss-simple-vars -D
```
## run
```
$ npm start
```

## build

```
$ npm run build
```
